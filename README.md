## Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/cmash93/Text-Editor.git
   ```
2. Install NPM packages
   ```sh
   npm i
   ```
4. To create your dist folder:
   ```sh
   npm run build
   ```
5. Start the server
   ```sh
   npm run start:dev
   ```
